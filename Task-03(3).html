Many APIs have moved toward supporting GraphQL in addition to REST or even supporting it exclusively. However, if you need to consume a GraphQL API, you wouldn't be blamed for thinking that you must use a JavaScript framework such as React or several additional libraries just to get it to work. That's because many tutorials and sample code seem to work off of the assumption that, if you're working with GraphQL, you are working with these libraries.

However, a query to a GraphQL API is just a properly formatted HTTP request. A GraphQL response is just JSON. You don't need any fancy libraries to handle either of these. In this tutorial, take a look at a different approach, and learn how easy it is to call a GraphQL API from both Node and client-side JavaScript without libraries..
